# Movie App

A simple app that provides movie information from <a href="https://developers.themoviedb.org/3/getting-started/introduction" rel="nofollow">TMDB API</a> .
<br />  
### Features:
<ul>  
  <li>Now Showing</li>
  <li>Popular</li>
  <li>Movie Detail</li>
  <li>Movie Trailer</li>
</ul>
